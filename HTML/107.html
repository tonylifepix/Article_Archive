# 深圳大学首届社区电影文化节，精彩不停！ ## 林子 ## 2015-04-09

        人间四月芳菲尽，深大春意正浓时！为进一步丰富和活跃深圳大学校园文化，利用社区资源拓宽学生课余文娱活动领域，营造一个积极健康的社区文化氛围，学生部学生社区管理中心将举办“深圳大学首届社区电影文化节”，2015年4月至5月将面向深圳大学在校学生、教职员工举行一系列的活动。

 

一、“荔园微故事”图片征集大赛

 

1.  活动时间：4月3日-4月20日

用镜头记录画面，让图片讲述故事。以“若干张图片串联起一个小故事”的主题叙事方式，通过镜头语言塑造人物、提炼情节，去记录描述荔园的人、景、事。

2.  作品要求：

1）创作题材及角度不限，内容积极健康；

2）图片张数在1-6张范围内，可配上文字介绍；

3）所有参赛作品必须为投稿者原创

3.  投稿方式：4月3日-4月9日发送作品至邮箱szusqb@163.com或深大社区微信公众号，学生以“图片大赛+作品命名+所在宿舍+学号+姓名+联系方式”作为参赛投稿标题；教职工以“图片大赛+作品命名+所在单位+姓名+联系方式”作为参赛投稿标题。

4.  评选方法：组织评委对参赛作品进行第一轮评比，评选出的15份优秀作品将进入第二轮线上全民投票。

5.  奖项设置：

一等奖1名：价值1000元的奖品及证书

二等奖1名：价值800元的奖品及证书

三等奖1名：价值500元的奖品及证书

（注：投稿者前100名可获得电影《左耳》明星见面会门票）

 

二、“寻找小耳朵”狂欢趴

        当“电影主题曲live”小型音乐会遇上形式新颖的户外摊位游戏。音乐会采用预报名和现场即兴两种参与方式，并根据现场观众的投票结果选出当晚的优秀选手。户外摊位游戏以“电影文化”为主线展开，参与摊位游戏可获积分奖励，累计一定积分可换取相应奖品。

1.  活动时间：4月12日晚上

2.  活动地点：西南篮球场

3.  音乐会参与方式：

1）即日起至4月9日发送“音乐会+所在宿舍+学号+姓名+联系方式+演唱曲目”到深大社区微信公众号或者编辑短信发送到15014000332/690332参与报名。

2）可选择个人或团队参赛。

3）自备伴奏（乐器、demo等）。

4.  选曲要求：电影主题曲

5.  评选方法：专业评委打分（60%）及现场观众投票（以给参赛选手贴“心”的方式）（40%）

6.  奖项设置：

冠军（1名）：奖品ipad mini及获奖证书

最佳声线奖（1名）：奖品kindle及获奖证书

最佳人气奖（1名）：奖品亚马逊尊贵黑卡（500元礼品卡）及获奖证书

最佳台风奖（1名）：奖品尤克里里及获奖证书

（注：参赛者均可获得电影《左耳》明星见面会门票，获奖者还有机会和五阿哥手牵着手牵着手牵着手？

另外，玩摊位游戏集印章可换精美小礼品，并有机会获得《左耳》明星见面会vip门票。快来召唤神龙吧！”

）


 

三、《左耳》高校行明星见面会

       学生社区管理中心承办青春电影《左耳》进校园活动，电影导演及剧组人员将来到深大校园，与同学们一起畅聊最美好的青春时光。

1.  活动时间：4月13日下午3点开始

2.  活动地点：演会中心

3.  领票方式：4月9日—10日11:30-13:30，两个时间段可到西南、学活派票点领取入场券，先到先得，领完即止。
